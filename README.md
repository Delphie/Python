# Python
All my Python projects in one repository.

## Scripts

### [birthday_reminder](birthday_reminder/birthday_reminder.py)
Looks up a birthday by name and prints the date.

### [hours_worked](hours_worked/hoursWorkedPayFunction.py)
Calculates weekly and monthly pay, including overtime (2x pay for hours over 40).

### [try_except](try_except/try_except_sample.py)
Demonstrates Python try/except error handling by summing two user-entered numbers.

### [password_checker](password_checker/password_checker.py)
Checks password strength and returns "too short", "Weak", "Medium", or "Strong".

### [drawing_turtle](drawing_turtle/shapes.py)
Uses Python's `turtle` module to draw colored shapes (hexagon, square, star).

### [python_datetime](python_datetime/date_time.py)
Demonstrates Python's `datetime` module with different date/time format examples.
